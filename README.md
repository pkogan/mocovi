# mocovi
Aplicación web para planificación y ejecución del presupuesto Universitario
Desarrollada sobre framework siu-toba.

# Módulo Presupuesto Decentralizado: tiene por objetivo generar el presupuesto desde cada dependencia. Cada referente debe cargar: los cargos, permutas y gastos que prevee para el año próximo.

Módulo Seguimiento Presupuestario: tiene por objetivo informar a cada dependencia y al rectorado el estado actual de su crédito de cargos y saldos. En desarrollo.
